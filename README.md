## What to do  
1. Clone this repo     
2. Run `npm install` to download the node_modules folder   
3. Run `npm start`
   
## Technologies ⚙️  
 
* HTML5   
* CSS3 
* React js


# Description
This is a react app to list recipes.
It allows users to create, update, and delete recipes.
